# 新手引导系统

> **状态**: 已完成
> **作者**: 开发者 + Claude
> **最后更新**: 2026-03-27

---

## Overview（概述）

新手引导系统在玩家首次进入游戏时提供分步骤的交互式教程，覆盖核心操作：移动英雄、放置动物塔、开始波次、使用技能。引导通过高亮目标区域 + 文字气泡的方式逐步引导，不强制接管输入（玩家仍可自由操作，但错误操作时提示等待）。引导完成后不再触发，可在设置中重置。

---

## Player Fantasy（玩家幻想）

好的引导是「我自己学会的」而不是「被教程拖着走的」。玩家按引导放下第一只猎豹、开始第一波、看到猎豹打死第一只迅猛龙——这个「成功体验」确立了游戏的基本反馈循环。引导的目标不是教会所有机制，而是让玩家在第一局结束时对「我应该做什么」有清晰认知，降低「不知道怎么玩」导致的流失。

---

## Detailed Design（详细设计）

### 引导触发条件

```
首次启动游戏（存档中 tutorial_completed == false）
自动进入引导局（强制使用关卡1，普通难度）
引导局与正常局使用相同游戏逻辑，只增加引导层
```

### 引导步骤序列（共8步）

```
Step 1：认识地图
  高亮：整个地图区域
  气泡文字：「这是战场！绿色区域可以放置动物守卫，
             红色路径是史前生物的入侵路线。」
  推进条件：玩家点击「知道了」按钮

Step 2：移动英雄
  高亮：英雄单位
  气泡文字：「点击地图任意位置，指挥你的老虎英雄移动！
             英雄靠近动物塔会提升它们的攻速。」
  推进条件：英雄成功移动到任意位置

Step 3：放置第一只塔
  高亮：底部建造面板中的「猎豹」按钮
  气泡文字：「点击猎豹，然后点击绿色建造格放置它！
             费用：50金币」
  推进条件：成功放置一只猎豹
  错误处理：玩家点击路径格时，提示「那里是敌人的路，
             试试绿色的草地格！」

Step 4：开始第一波
  高亮：「开始波次」按钮
  气泡文字：「准备好了！点击开始，迎接第一波史前生物。」
  推进条件：玩家点击开始波次按钮

Step 5：观察战斗
  高亮：无（全场自由观察）
  气泡文字：「看！猎豹开始攻击了。消灭所有敌人，
             不要让它们到达右下角的基地！」
  推进条件：第1波结束（所有敌人死亡或到达终点）

Step 6：使用技能
  高亮：技能栏中的「百兽怒吼」按钮（Q键）
  气泡文字：「英雄有三个强力技能！现在试试百兽怒吼——
             让全场动物塔攻速爆发！」
  推进条件：玩家使用百兽怒吼技能

Step 7：认识商店（若第3波后触发）
  高亮：商店界面（第3波后自动弹出）
  气泡文字：「商店开门了！用金币购买道具强化你的阵容，
             或点击刷新获得新的选项。」
  推进条件：关闭商店界面

Step 8：引导完成
  气泡文字：「你已经掌握了基础！接下来靠你自己了。
             祝你好运，守护者！」
  按钮：「开始冒险」
  推进条件：玩家点击按钮
  后续：标记 tutorial_completed = true，存档保存
```

### 引导UI规范

```
高亮方式：目标区域正常显示，其余区域半透明暗色遮罩（alpha=0.6）
气泡位置：自动避开高亮区域（高亮在下方则气泡显示在上方）
气泡样式：像素风对话框，带角色头像（老虎英雄头像）
跳过按钮：右上角「跳过引导」，点击直接完成所有步骤
```

### 引导数据

```
TutorialState {
  tutorial_completed: bool   // false=未完成，首次启动触发引导
  current_step: int          // 当前引导步骤（0~8）
}
// 存储于存档系统
```

---

## Formulas（公式）

**引导完成时间估算**
```
8个步骤，每步平均30秒（含操作时间）
总引导时长：约4~6分钟
目标：不超过5分钟（避免新玩家因引导过长而流失）
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 引导中途关闭游戏 | 下次启动从 current_step 继续（不从头开始） |
| 玩家在Step 3前金币不足放塔 | 引导局初始金币保证≥100（足够放一只猎豹），不会出现此情况 |
| 玩家在Step 4前提前点击开始波次 | 允许，Step 4标记为已完成，跳到Step 5 |
| Step 6要求使用技能但玩家不操作 | 等待，气泡持续显示，无超时强制推进 |
| 引导完成后玩家想重新观看 | 设置菜单中「重置新手引导」按钮，将tutorial_completed重置为false |
| 跳过引导 | 所有步骤标记完成，tutorial_completed=true，立即进入正常游戏 |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| 引导步骤总数 | 8步 | 5~12步 | 越少引导越简洁但覆盖机制越少，越多学习越完整但时间越长 |
| 遮罩透明度 | 0.6 | 0.4~0.8 | 越高焦点越集中，越低游戏世界可见度越高 |
| 引导局初始金币 | 100 | 80~150 | 确保玩家能完成至少一次放塔操作 |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 首次启动自动进入引导，第二次启动不触发 | 新存档启动确认引导触发；完成后重启确认不再触发 |
| AC-2 | 每个步骤高亮正确目标区域 | 逐步验证：Step 3时确认猎豹按钮高亮，其余界面半透明 |
| AC-3 | Step 3点击路径格时显示错误提示 | 在引导Step 3点击路径格，确认提示文字出现 |
| AC-4 | 引导中途关闭重启后从中断步骤继续 | Step 5时关闭游戏，重启后确认从Step 5继续 |
| AC-5 | 跳过引导直接进入正常游戏 | 点击「跳过引导」，确认立即进入正常游戏且不再触发引导 |
| AC-6 | 设置中重置引导后下次启动重新触发 | 重置后重启，确认引导从Step 1开始 |
