# 图鉴收集系统

> **状态**: 已完成
> **作者**: 开发者 + Claude
> **最后更新**: 2026-03-27

---

## Overview（概述）

图鉴收集系统记录玩家在游戏中「首次遭遇」的动物、敌人和道具，并在图鉴界面展示其属性和背景故事。每个条目首次解锁时给予少量古代印记奖励。图鉴是 Alpha 阶段的养成钩子——完成度进度条驱动玩家尝试不同的 Build 和关卡，即使已经通关也有回来的理由。

---

## Player Fantasy（玩家幻想）

图鉴满足「收集癖」——看着图鉴从灰色问号一格格点亮，是一种独立于战斗的满足感。每次用到新动物或遭遇新敌人，图鉴更新的提示让玩家知道「这局做了新的事情」。背景故事文本（几句话的世界观描述）让动物和敌人不只是数值，而是有个性的角色，增加游戏的情感厚度。

## Detailed Design（详细设计）

### 图鉴条目分类

```
动物图鉴：12条（对应12只动物）
敌人图鉴：10条（对应10种敌人）
道具图鉴：20条（对应20种道具）
总计：42条
```

### 解锁触发条件

```
动物图鉴：首次将该动物放置到地图上
敌人图鉴：首次击杀该敌人（HP归零）
道具图鉴：首次获得该道具（进入背包）

解锁时：
  1. 图鉴条目从「???」变为正常显示
  2. 屏幕右下角弹出「[图鉴] 新条目解锁：[名称]」（2秒）
  3. 给予+3古代印记奖励
```

### 图鉴条目内容

```
每个条目包含：
  - 单位像素艺术图（64×64）
  - 名称（调用本地化系统）
  - 类型标签（动物/史前生物/道具）
  - 核心属性展示（HP、攻速、特殊效果等）
  - 背景故事（2~3句话，体现世界观）
  - 首次解锁日期

未解锁条目显示：灰色轮廓 + 「???」
```

### 图鉴界面结构

```
入口：主菜单「图鉴」按钮
布局：
  顶部：完成进度「已解锁 X / 42」+ 进度条
  三个标签页：动物 / 敌人 / 道具
  网格展示：每行4个条目，点击查看详情
  详情面板：右侧弹出，显示完整条目信息
```

### 数据结构

```
Collection {
  unlocked_entries: string[]   // 已解锁条目ID列表
  unlock_dates: {[id]: string} // 各条目首次解锁日期
}
// 存储于存档系统 collection 字段
```

## Formulas（公式）

**完整收集所需局次估算**
```
每局平均新解锁：
  动物：2~3只新动物（早期局次）
  敌人：3~5只新敌人（前几局）
  道具：2~3件新道具（商店随机）
平均每局新解锁 ≈ 7~10条
42条全部解锁约需 5~7 局（鼓励多局游玩）
```

**图鉴印记总奖励**
```
42条 × 3印记 = 126印记
约占图腾神殿总费用（410印记）的30%
图鉴是印记的重要补充来源
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 敌人在未被击杀时到达终点（漏过）| 不解锁图鉴（需要击杀才触发）|
| 道具在背包满时无法进入，是否解锁 | 不解锁（道具未实际获得）|
| 已解锁条目再次触发 | 忽略，不重复给印记 |
| Alpha新增动物/敌人/道具与旧存档兼容 | 新条目默认未解锁，旧存档不受影响 |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| 图鉴总条目数 | 42 | 30~60 | 越多收集目标越丰富，内容量要求越高 |
| 解锁印记奖励 | 3印记/条 | 1~5印记 | 越高图鉴对跨局成长的贡献越大 |
| 解锁弹出提示时长 | 2秒 | 1~3秒 | 越长玩家越容易看到，越短越不打扰 |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 首次放置猎豹解锁猎豹图鉴条目并给予+3印记 | 首局放置猎豹，确认图鉴更新提示弹出，印记+3 |
| AC-2 | 已解锁条目再次触发不重复给印记 | 第二局再次放置猎豹，确认无弹出提示，印记不增加 |
| AC-3 | 未解锁条目显示灰色轮廓和「???」 | 打开图鉴，确认未遭遇的敌人显示为问号 |
| AC-4 | 图鉴进度条正确显示解锁数量 | 解锁10条后，确认进度显示「10 / 42」 |
| AC-5 | 图鉴数据在游戏重启后保持 | 解锁若干条目后重启游戏，确认图鉴状态保留 |
| AC-6 | 道具背包满时获得道具不解锁图鉴 | 背包6格满时商店购买新道具失败，确认该道具图鉴未解锁 |
