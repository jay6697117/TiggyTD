# 英雄技能树系统

> **状态**: 已完成
> **作者**: 开发者 + Claude
> **最后更新**: 2026-03-27

---

## Overview（概述）

英雄技能树系统在局内提供老虎英雄的被动成长路径。玩家通过击杀敌人获得经验值（EXP），英雄升级后选择一个被动天赋强化，每局最多升3级（共选3个天赋）。天赋影响英雄的技能效果、光环范围或战场控制能力，与动物技能树共同构成每局 Build 的差异化来源。

---

## Player Fantasy（玩家幻想）

英雄升级的瞬间是「我变强了」的直觉满足。在最关键的波次前恰好升级、选到「自然之召」强化天赋，然后站在最密集的塔群中心放大招——这种「天时地利」的感觉是英雄成长感的精髓。天赋选择让每局的英雄都是「这一局专属的英雄」，而不只是一个通用工具。

## Detailed Design（详细设计）

### 升级机制

```
英雄初始等级：1级
最高等级：4级（共升3次）
每次升级：弹出天赋选择面板，展示3个随机天赋选项（从天赋池中随机抽取）
选择后：天赋立即生效，面板关闭

EXP获取：
  普通敌人死亡：+5 EXP
  精英敌人死亡：+15 EXP
  Boss死亡：+50 EXP

EXP升级阈值：
  1→2级：50 EXP
  2→3级：120 EXP
  3→4级：250 EXP
```

### 天赋池（共12个天赋，每次从中随机3个展示）

```
【技能强化类】
  猛虎咆哮强化：百兽怒吼持续时间 +4秒（8→12秒）
  猛虎咆哮扩散：百兽怒吼冷却时间 -10秒（30→20秒）
  自然之召强化：自然之召攻击加成 +10%（30%→40%）
  自然之召扩散：自然之召范围 +2格（5格→7格）
  纪元审判强化：纪元审判伤害 +150（300→450）
  纪元审判扩散：纪元审判冷却 -30秒（120→90秒）

【光环强化类】
  光环扩展：英雄光环半径 +2格（3→5格）
  光环增幅：英雄光环攻速加成 +10%（20%→30%）

【被动战场类】
  迅捷图腾：英雄移动速度 +30%
  战场感知：英雄视野范围 +3格（无战争迷雾的MVP中等于装饰效果，Alpha生效）
  黄金触碰：每击杀10只敌人额外获得5金币
  韧性图腾：基地HP上限 +2（10→12）
```

### 天赋随机规则

```
每次升级展示3个天赋：
  从12个天赋中随机抽取3个（不重复）
  已选过的天赋从池中移除（不会重复选到同一天赋）
  若池中剩余天赋 < 3，展示全部剩余天赋
```

### 升级UI流程

```
1. 英雄EXP达到阈值 → 触发升级
2. 游戏暂停（time_scale = 0）
3. 弹出天赋选择面板（全屏半透明遮罩）
4. 展示3个天赋卡片（图标 + 名称 + 效果描述）
5. 玩家点击一张卡片
6. 天赋立即应用，游戏恢复（time_scale = 1）
7. 英雄EXP条归零，开始积累下一级经验
```

## Formulas（公式）

**EXP进度**
```
1→2级：50 EXP（约消灭10只普通敌人）
2→3级：120 EXP（约消灭24只普通敌人）
3→4级：250 EXP（约消灭50只普通敌人）
总计10波约200~300只普通敌人，玩家通常能升满4级
```

**光环扩展天赋后覆盖面积**
```
原半径3格：面积 ≈ π×9 ≈ 28格
扩展后半径5格：面积 ≈ π×25 ≈ 78格
覆盖面积提升约179%
```

**黄金触碰天赋收益**
```
每10击杀+5金币
10波约250只敌人 → 额外+125金币
约等于额外2只中等动物塔的费用
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 天赋选择面板弹出时正在施法 | 游戏暂停（time_scale=0），施法动画冻结，选完天赋后继续 |
| 同一波次内连续升级两次 | 第一次选择完毕后立即检查EXP，若已满再次弹出第二次天赋面板 |
| 剩余天赋少于3个时 | 展示全部剩余天赋（1~2个），玩家必须选一个 |
| 天赋池全部选完（不可能，最多选3次，池有12个） | 理论上不会发生，无需处理 |
| 韧性图腾选取时基地HP已扣过 | base_hp_max +2，当前base_hp不回复（只提升上限，不治疗） |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| 升级EXP阈值（1→2） | 50 EXP | 30~80 | 越低玩家越早获得天赋，成长感越快 |
| 升级EXP阈值（3→4） | 250 EXP | 150~350 | 越高玩家越晚满级，最终强度影响后期波次 |
| 普通敌人EXP | 5 | 3~10 | 影响升级速度 |
| 天赋展示数量 | 3个 | 2~4个 | 越多选择越多但决策负担越重 |
| 光环扩展天赋半径增量 | +2格 | +1~+3格 | 影响光环覆盖面积增幅 |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 击杀普通敌人获得5 EXP | 击杀一只迅猛龙，确认EXP条增加5点 |
| AC-2 | EXP达到50时触发升级，弹出天赋面板 | 积累50 EXP，确认游戏暂停并弹出3张天赋卡 |
| AC-3 | 天赋面板展示3个不重复选项 | 多次升级，确认每次3个选项互不相同 |
| AC-4 | 选择天赋后效果立即生效 | 选择「光环扩展」，确认英雄光环圆圈视觉立即扩大到5格 |
| AC-5 | 已选天赋不再出现在后续选项中 | 第1次选了「猛虎咆哮强化」，第2次升级确认该天赋不再出现 |
| AC-6 | 选择「韧性图腾」后base_hp_max变为12 | 选择该天赋，确认基地HP上限显示12（若当前HP已扣，不回复） |
| AC-7 | 天赋选择期间游戏世界暂停 | 弹出天赋面板时，确认敌人停止移动、动物塔停止攻击 |
