# 协同效果系统

> **状态**: 已完成
> **作者**: 开发者 + Claude
> **最后更新**: 2026-03-27

---

## Overview（概述）

协同效果系统检测场上动物塔的组合，当特定组合同时存在时激活额外的被动加成。协同是 Build 深度的最高层表达——不只是单只动物强，而是「这些动物放在一起会产生化学反应」。系统每次塔放置/移除时重新计算所有协同条件，满足条件的协同立即生效，不满足时立即失效。

---

## Player Fantasy（玩家幻想）

协同的高光时刻是「发现组合」：玩家在商店买了第三只鬣狗，突然全场鬣狗攻速爆涨、屏幕出现「狂野猎手 ×3」的金色提示——那种「我发现了秘密」的感觉是 Build 游戏最强的钩子。协同让玩家主动追求特定动物组合，而不是随机放塔，把「攒够3只」变成清晰的短期目标。

---

## Detailed Design（详细设计）

### MVP 协同列表（共6个）

```
1. 狂野猎手（Hyena Trio）
   条件：场上 ≥3只鬣狗
   效果：所有鬣狗攻速 +30%

2. 铁壁防线（Turtle Wall）
   条件：场上 ≥2只乌龟
   效果：所有乌龟护甲效果 ×1.5（减伤加倍）

3. 闪电突袭（Cheetah Pack）
   条件：场上 ≥2只猎豹
   效果：所有猎豹攻击附加10%概率「眩晕」0.5秒

4. 地动山摇（Elephant Herd）
   条件：场上 ≥2只大象
   效果：所有大象踩踏附加「减速50%持续2秒」

5. 丛林法则（Predator Mix）
   条件：场上同时有猎豹+鬣狗+狼群
   效果：三种动物攻击伤害 +15%

6. 钢铁意志（Tank Duo）
   条件：场上同时有乌龟+大象
   效果：乌龟和大象获得「嘲讽光环」——周围3格敌人优先攻击它们（若有攻击行为）
```

### 协同计算逻辑

```
on_tower_placed(tower) 或 on_tower_removed(tower):
  recalculate_all_synergies()

recalculate_all_synergies():
  active_towers = get_all_placed_towers()
  count_by_type = group_by(active_towers, key=animal_type)

  for each synergy in SYNERGY_DEFINITIONS:
    was_active = synergy.is_active
    now_active = synergy.check_condition(count_by_type)

    if now_active and not was_active:
      synergy.is_active = true
      apply_synergy_effects(synergy)
      show_synergy_activated_vfx(synergy)

    elif not now_active and was_active:
      synergy.is_active = false
      remove_synergy_effects(synergy)
      show_synergy_deactivated_vfx(synergy)
```

### 协同效果叠加规则

```
协同加成独立于技能树加成和道具加成，三者加法叠加后再乘以光环
示例：猎豹闪电突袭协同（眩晕概率10%）+技能树B2（减速30%）
  → 攻击有10%概率眩晕，有30%概率减速（独立触发，可同时生效）

同类属性多个协同：加法叠加
  若将来两个协同同时给鬣狗攻速加成：+30% + X% = +(30+X)%
```

### 协同UI展示

```
战斗界面右侧：「协同」面板
  列出所有已激活协同：图标 + 名称 + 效果描述
  未激活协同：显示灰色 + 「还需X只[动物类型]」
协同激活时：屏幕中央短暂弹出金色文字「[协同名] 已激活！」（1.5秒）
```

## Formulas（公式）

**协同强度设计原则**
```
协同额外收益 = 10~30%性能提升（在已有技能树+道具基础上）
不能过强（否则玩家只追协同，放弃其他策略）
不能过弱（否则协同感知不到，失去意义）
目标：激活协同比不激活 DPS 提升约15~25%
```

**狂野猎手协同DPS提升**
```
3只鬣狗各自攻速+30%
期望DPS提升 = 30%（线性，攻速直接等比提升DPS）
3只鬣狗总DPS提升 = +30%（相对于无协同状态）
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 三角龙将第3只鬣狗推出场地（不是消灭） | 下次recalculate时鬣狗数=2，狂野猎手协同失效 |
| 波次进行中放置第3只鬣狗激活协同 | 立即激活，攻速立即提升，无延迟 |
| 多个协同同时激活（如放置乌龟同时满足铁壁+钢铁意志） | 两个协同均激活，效果均生效，UI同时显示两条激活提示 |
| 协同涉及的动物被升级技能树 | 协同效果基于运行时属性，技能树修改后协同加成也作用于新属性 |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| 狂野猎手攻速加成 | +30% | +15~50% | 越高追求鬣狗协同的收益越大 |
| 闪电突袭眩晕概率 | 10% | 5~25% | 越高猎豹组合控制力越强 |
| 丛林法则伤害加成 | +15% | +10~25% | 跨种族协同的混搭奖励 |
| 协同激活最低单位数 | 2~3只 | 2~4只 | 越少越容易触发，协同泛滥；越多稀缺感越强 |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 放置第3只鬣狗激活「狂野猎手」，所有鬣狗攻速+30% | 放置2只鬣狗后放第3只，确认3只鬣狗攻速均提升30% |
| AC-2 | 移除一只鬣狗后协同失效，攻速恢复 | 卖掉一只鬣狗（若有卖塔功能），确认剩余2只攻速恢复原值 |
| AC-3 | 协同激活时屏幕弹出金色提示1.5秒 | 触发任意协同，确认激活提示文字出现并1.5秒后消失 |
| AC-4 | 协同面板显示已激活和未激活协同 | 确认右侧面板中激活协同高亮，未激活协同灰色显示缺少条件 |
| AC-5 | 丛林法则需猎豹+鬣狗+狼群同时在场 | 只放猎豹+鬣狗确认未激活；加入狼群后确认激活 |
| AC-6 | 协同效果与技能树加成正确叠加 | 猎豹技能树A(+15%攻速)+闪电突袭协同(眩晕)同时生效，两者不互斥 |
