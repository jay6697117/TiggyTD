# 跨局成长系统

> **状态**: 已完成
> **作者**: 开发者 + Claude
> **最后更新**: 2026-03-27

---

## Overview（概述）

跨局成长系统提供局外持久化成长机制：玩家完成游戏局次后获得「古代印记」（硬货币），可在主菜单的「图腾神殿」中永久解锁被动加成或初始福利。这些解锁在所有后续局次中生效，是 Roguelite 长线留存的核心钩子。跨局进度存储在存档系统中，不会因局内失败而重置。

---

## Player Fantasy（玩家幻想）

跨局成长让失败也有意义——每局结束都有「古代印记」入账，积累解锁新的永久加成，下一局从更强的起点出发。「再来一局」的冲动来自「我又多了10个印记，可以解锁那个初始金币加成了」。这种「每局都在变强」的感知是 Roguelite 反复游玩的根本驱动力。

## Detailed Design（详细设计）

### 古代印记获取

```
每局结束后结算：
  通关（胜利）：+30 古代印记
  失败但撑过第5波：+10 古代印记
  失败在第5波前：+5 古代印记
  首次通关某关卡：额外 +20 古代印记（一次性奖励）
  Boss击杀：+15 古代印记（每局最多一次）
```

### 图腾神殿（解锁树，共12个节点）

```
【经济类】
  初始金币+20（费用：15印记）
  商店刷新折扣：刷新费用 30→25金币（费用：25印记）
  波次击杀金币+1（费用：30印记）

【战斗类】
  初始基地HP+2（费用：20印记）
  英雄初始移速+10%（费用：15印记）
  英雄技能初始CD-10%（费用：30印记）

【Build类】
  动物塔技能树费用-10%（费用：25印记）
  商店初始第一件道具免费（费用：40印记）
  初始背包+1格（7格）（费用：20印记）

【解锁类】
  解锁第2张地图（费用：50印记）
  解锁困难难度（费用：60印记）
  解锁新英雄（Alpha预留，费用：80印记）
```

### 解锁规则

```
所有节点独立解锁（无前置依赖树，玩家自由选择购买顺序）
每个节点只能购买一次（不可叠加）
购买后立即对下一局生效
退款：不支持（永久解锁）
```

### 数据结构

```
MetaProgression {
  ancient_marks: int              // 当前持有的古代印记数
  total_marks_earned: int         // 历史总获得印记数
  unlocked_nodes: string[]        // 已解锁的节点ID列表
}
// 存储于存档系统 meta_progression 字段
```

## Formulas（公式）

**首次完整解锁所有节点所需印记**
```
总费用 = 15+25+30+20+15+30+25+40+20+50+60+80 = 410 印记
平均每局收益（含胜利+Boss）≈ 30+15 = 45 印记/局（胜利局）
失败局平均 ≈ 7.5 印记
假设胜率30%：平均每局 = 0.3×45 + 0.7×7.5 ≈ 18.75 印记
全部解锁约需 410/18.75 ≈ 22 局
```

**跨局加成对难度的影响**
```
初始金币+20：相当于提前拥有半只普通动物塔的资金
初始基地HP+2：错误容忍度提升20%（10→12）
目标：全解锁状态下通关率从30%提升至约60%
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 古代印记不足时点击解锁节点 | 拒绝，提示「古代印记不足，还需X个」 |
| 存档损坏导致解锁数据丢失 | 古代印记归零，已解锁节点清空；显示错误提示，不强制回档 |
| 同一关卡多次通关 | 首次通关额外奖励只发放一次，后续通关只计基础30印记 |
| Alpha新节点加入后旧存档兼容 | 新节点默认未解锁，旧存档印记数保持，不影响已解锁节点 |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| 胜利印记奖励 | 30 | 20~50 | 越多玩家解锁速度越快，长线留存压缩 |
| 失败印记奖励 | 5~10 | 3~15 | 越多失败也有明显收益，降低挫败感 |
| 最昂贵节点费用 | 80印记 | 50~120 | 决定「终极目标」的追求时长 |
| 解锁节点总数 | 12 | 8~20 | 越多长线目标越丰富，设计量也越大 |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 胜利局结算后获得30古代印记 | 通关一局，确认结算界面显示+30印记，总印记增加 |
| AC-2 | 失败但过第5波获得10印记 | 在第6波失败，确认获得10印记 |
| AC-3 | 图腾神殿正确显示所有节点和费用 | 打开神殿，确认12个节点全部显示，费用正确 |
| AC-4 | 解锁「初始金币+20」后下一局开局金币增加 | 解锁该节点，开始新局，确认初始金币从默认值+20 |
| AC-5 | 同一节点不可重复购买 | 已解锁节点变灰，再次点击无反应 |
| AC-6 | 跨局进度在游戏重启后保持 | 解锁节点后关闭游戏重启，确认解锁状态保留 |
