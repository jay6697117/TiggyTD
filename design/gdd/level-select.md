# 关卡选择/进度系统

> **状态**: 已完成
> **作者**: 开发者 + Claude
> **最后更新**: 2026-03-27

---

## Overview（概述）

关卡选择/进度系统管理多关卡的解锁流程和主菜单的关卡选择界面。MVP阶段只有1张地图，Alpha阶段扩展为3张。关卡按线性顺序解锁：通关当前最高关卡后解锁下一关。每个关卡记录最高通关状态（未解锁/已解锁/已通关），支持玩家重复挑战已通关关卡（刷印记或追求更高评分）。

---

## Player Fantasy（玩家幻想）

关卡选择界面是「我还有多少路要走」的进度感来源。看到第2关从锁定变为可选、点进去面对全新地图布局——这种「新内容解锁」的时刻是长线游玩的里程碑。每张地图的差异化（不同路径形状、不同地形约束）让同样的 Build 策略需要适应新环境，保持新鲜感。

## Detailed Design（详细设计）

### 关卡定义（Alpha阶段共3关）

```
关卡1：远古草原（Ancient Savanna）
  地图：S形双弯道（20×15格）
  解锁条件：默认解锁
  主题：开阔草原，建造空间充足

关卡2：熔岩峡谷（Lava Canyon）
  地图：Z形三折道（20×15格，建造空间较少）
  解锁条件：通关关卡1
  主题：火山地形，部分格子为熔岩（无法建造）

关卡3：冰封苔原（Frozen Tundra）
  地图：螺旋形路径（20×15格，路径最长）
  解锁条件：通关关卡2
  主题：冰雪地形，敌人初始移速+10%
```

### 关卡进度数据

```
LevelProgress {
  level_id: string
  status: enum  // LOCKED / UNLOCKED / CLEARED
  best_result: {
    waves_survived: int    // 最高撑过波次
    base_hp_remaining: int // 通关时剩余基地HP（-1=未通关）
  }
}
// 存储于存档系统 level_progress[] 字段
```

### 解锁逻辑

```
on_game_win(level_id):
  level = get_level(level_id)
  level.status = CLEARED
  level.best_result.base_hp_remaining = max(当前值, 本局剩余HP)

  next_level = get_next_level(level_id)
  if next_level and next_level.status == LOCKED:
    next_level.status = UNLOCKED
    show_unlock_notification(next_level)  // 「新关卡解锁：熔岩峡谷！」
```

### 关卡选择UI

```
入口：主菜单「开始游戏」→ 关卡选择界面
布局：
  横向排列3个关卡卡片
  已通关：显示金色星形 + 剩余基地HP记录
  已解锁未通关：正常显示，可点击进入
  未解锁：灰色锁定图标 + 「通关[前置关卡]以解锁」

关卡卡片信息：
  关卡名称、地图缩略图、难度标签、最佳记录
```

### 难度选择（跨局成长系统解锁困难难度后）

```
普通难度（默认）：正常敌人属性
困难难度（需解锁）：
  敌人HP ×1.5
  敌人移速 ×1.2
  基地初始HP=7（而非10）
```

## Formulas（公式）

**关卡解锁进度**
```
线性解锁：关卡1→关卡2→关卡3
解锁关卡2需通关关卡1（胜利条件：消灭第10波全部敌人）
解锁关卡3需通关关卡2
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 已通关关卡再次进入并失败 | 关卡状态保持CLEARED，best_result不更新（只更新更好的记录） |
| 跨局成长解锁「第2张地图」与线性解锁冲突 | 跨局解锁等同于通关前置关卡——强制解锁关卡2，不要求通关关卡1 |
| 存档损坏导致关卡进度丢失 | 所有关卡重置为初始状态（只有关卡1可进入） |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| Alpha关卡总数 | 3 | 3~5 | 越多内容量越丰富，设计量也更大 |
| 困难难度敌人HP倍率 | ×1.5 | ×1.2~×2.0 | 越高困难难度挑战性越强 |
| 困难难度基地HP | 7 | 5~9 | 越低容错越小，难度越高 |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 游戏初始只有关卡1可进入，关卡2/3锁定 | 新存档打开关卡选择，确认关卡2/3灰色锁定 |
| AC-2 | 通关关卡1后关卡2解锁并弹出提示 | 通关关卡1，确认「新关卡解锁：熔岩峡谷」提示出现 |
| AC-3 | 关卡通关记录显示最佳剩余基地HP | 以8HP通关，记录显示「最佳：8HP」；再以6HP通关，记录保持8HP |
| AC-4 | 困难难度敌人HP为普通的1.5倍 | 困难难度下检查迅猛龙HP=150（普通100×1.5） |
| AC-5 | 跨局解锁「第2张地图」后关卡2强制解锁 | 购买该跨局节点，确认关卡2变为可进入 |
