> **状态**: 已完成

---

## Overview（概述）

Boss 系统定义第10波（MVP）的特殊敌人——沧龙霸主（Mosasaurus Rex）的登场、行为、特殊机制和死亡处理。Boss 是整局游戏的最终考验，拥有远超普通敌人的HP和特殊阶段技能。Boss 死亡触发胜利判定，穿越基地则扣3点HP且大概率直接导致失败。Boss 系统依赖敌人AI系统的基础框架，在其上叠加阶段切换和特殊技能逻辑。

---

## Player Fantasy（玩家幻想）

Boss 战是整局的情绪高潮——前9波精心构建的 Build 迎来最终检验。沧龙霸主缓缓登场，HP条巨大而压迫，第一阶段稳步推进……50%血线时护盾激活，全场一片紧张。纪元审判打破护盾的瞬间是最爽的操作时刻。最后一击落下，金色粒子爆开，胜利音效响起——这种「我赢了」的释放是整局所有准备和决策的情感回报。

## Detailed Design（详细设计）

### Boss 属性（沧龙霸主）

```
Boss ID：mosasaurus_rex
最大HP：2000
移动速度：0.8（比普通敌人慢，单位：格/秒）
护甲：20%
base_damage_to_base：3（穿越基地扣3点HP）
体型：3×2格（大型单位，占据3格宽2格高）
攻击能力：无（Boss只移动，不攻击动物塔）
```

### Boss 阶段设计

```
阶段一（100% → 50% HP）：正常移动，无特殊技能

阶段二（50% HP 触发）：
  触发条件：HP首次降至 ≤ 1000
  激活「远古护盾」：
    - Boss 获得无敌状态（免疫所有伤害）
    - 护盾视觉：蓝色光球包裹Boss
    - 护盾持续时间：8秒，或被纪元审判打破
    - 护盾破解：英雄使用「纪元审判」对Boss施放 → 护盾立即消除，Boss恢复受伤状态
    - 若8秒内未被打破：护盾自动消失，Boss继续受到伤害（但速度提升20%）
    - 护盾期间Boss继续正常移动（不停止）

阶段三（护盾消失后 → 死亡）：
  若护盾被纪元审判打破：Boss移动速度不变
  若护盾自然消失：Boss移动速度 × 1.2
  Boss HP降至0：触发胜利流程
```

### Boss 登场流程

```
第10波开始：
1. 波次系统触发 spawn_boss()
2. Boss从刷怪点出现（与普通敌人相同入口）
3. 登场动画：Boss出现时播放2秒震地动画（屏幕轻微震动）
4. 全屏提示：「沧龙霸主降临！」（显示2秒后消失）
5. 警告音效播放
6. Boss沿路径正常移动
```

### Boss 死亡处理

```
boss_die():
1. 播放死亡动画（3秒爆炸特效）
2. 掉落5件道具（道具ID由关卡配置定义，MVP阶段固定掉落）
3. 调用 wave_system.on_boss_killed() → 触发胜负判定检查
4. 给予大量EXP：+200 EXP（若英雄未满级则大概率升级）
```

### Boss 血条UI

```
位置：屏幕顶部中央（独立于普通敌人血条）
显示：「沧龙霸主」名称 + 当前HP / 最大HP + 阶段图标
护盾激活时：血条变为蓝色，显示「护盾中」文字
```

## Formulas（公式）

**Boss 伤害压力**
```
Boss HP=2000，护甲20%
需要对Boss造成2000实际伤害才能击杀
考虑护甲：需要 2000 / (1 - 0.2) = 2500 点原始伤害
护盾阶段免疫伤害（最多8秒），假设全场DPS=200/秒
护盾损失时间成本：8秒 × 200 DPS = 1600点伤害损失
→ 玩家有动力使用纪元审判打破护盾
```

**护盾阶段Boss位移**
```
Boss移动速度=0.8格/秒
护盾持续8秒内移动距离：0.8 × 8 = 6.4格
地图路径总长约40格
护盾阶段Boss推进约16%路程
若在路径中段触发护盾，有足够时间打破
```

**Boss穿越扣血**
```
base_damage_to_base = 3
MVP base_hp_max = 10
一只Boss穿越 = 剩余基地HP × 30%被消耗
若基地HP=3，Boss穿越直接导致失败
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 护盾激活时玩家使用纪元审判对Boss以外的目标 | 护盾只响应对Boss直接施放的纪元审判，其他目标不打破护盾 |
| 护盾激活时Boss被推入陷阱（碰撞类伤害） | 护盾免疫所有伤害来源，包括碰撞和AOE |
| Boss到达终点时护盾仍激活 | Boss扣3点基地HP，护盾不影响穿越判定 |
| 阶段二护盾触发时Boss HP恰好=1000 | 触发条件为「首次降至≤1000」，精确等于1000也触发 |
| 护盾被打破的同帧Boss HP归零 | 死亡优先，直接触发死亡流程，不进入阶段三 |
| 纪元审判CD未就绪时护盾激活 | 玩家需等待CD，或接受护盾8秒后自然消失（Boss加速） |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| Boss最大HP | 2000 | 1500~3000 | 越高Boss战持续时间越长，对DPS要求越高 |
| 护盾触发阈值 | 50% HP | 30~70% HP | 越早触发护盾，玩家越需要提前准备纪元审判 |
| 护盾持续时间 | 8秒 | 5~15秒 | 越长玩家越难靠「等待」解决，强制使用技能 |
| 护盾破解后Boss加速 | ×1.2 | ×1.0~×1.5 | 护盾自然消失的惩罚程度 |
| Boss移动速度 | 0.8格/秒 | 0.5~1.2格/秒 | 越慢玩家有越多时间输出，Boss战压力越小 |
