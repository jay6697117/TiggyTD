# 动物技能树系统

> **状态**: 已完成
> **作者**: 开发者 + Claude
> **最后更新**: 2026-03-27

---

## Overview（概述）

动物技能树系统允许玩家在战斗中用金币为已放置的动物塔升级技能节点，提升其战斗能力或解锁新机制。每种动物有独立的2级技能树（3个节点，分支结构），玩家在一局内做出的选择不可撤销，构成 Build 差异化的核心来源。技能树节点由动物数据库定义，运行时由本系统管理解锁状态和效果应用。

---

## Player Fantasy（玩家幻想）

技能树是「这只猎豹是我培养的」的身份感来源。同一只猎豹，选择「暴击强化」和「攻速强化」会打出完全不同的战斗风格。当多只动物的技能树选择形成协同（猎豹暴击 + 鬣狗破甲 + 大象AOE），玩家看到那一局独特的 Build 发挥出来时，是最强的成就感时刻。技能树让每局不同，是 Roguelite 重复游玩动力的核心。

## Detailed Design（详细设计）

### 技能树结构

```
每只动物：3个技能节点，分支结构

节点布局：
  [节点A：基础强化]  ← 第1级（必须先解锁）
       ↓
  [节点B1：攻击路] 或 [节点B2：辅助路]  ← 第2级（二选一）

解锁条件：
  节点A：无前置要求，直接购买
  节点B1/B2：必须先解锁节点A
  同一动物B1和B2互斥（选一不可选另一）
```

### 各动物技能树定义

```
猎豹（Cheetah）
  A：攻速+15%（费用：40g）
  B1（攻击路）：暴击率+20%，暴击伤害×2.0（费用：60g）
  B2（辅助路）：命中目标有30%概率施加减速2秒（费用：60g）

乌龟（Turtle）
  A：护甲强化 — 自身受击伤害-50%（费用：30g）
  B1（坚守路）：每5秒向周围2格敌人释放减速光环（费用：50g）
  B2（反弹路）：受击时反弹10点真实伤害给攻击者（费用：50g）

鬣狗（Hyena）
  A：群体攻击 — 攻击溅射到目标周围1格（费用：50g）
  B1（嘲讽路）：吸引附近3格内所有敌人优先攻击自己（费用：70g）
  B2（破甲路）：攻击降低目标armor×0.5，持续5秒（费用：70g）

狼群（Wolf Pack）
  A：召唤第4只狼（从3只增加到4只）（费用：60g）
  B1（群狼路）：每只狼攻击+20%（费用：80g）
  B2（协调路）：3只以上狼同时攻击同一目标时，触发「群殴」+50%伤害（费用：80g）

大象（Elephant）
  A：踩踏范围+1格（从2格增加到3格）（费用：50g）
  B1（震慑路）：踩踏附加恐惧效果2秒（费用：80g）
  B2（推进路）：踩踏将敌人击退2格（费用：80g）
```

### 运行时数据结构

```
TowerSkillState {
  tower_id: int
  animal_type: string
  unlocked_nodes: string[]   // 已解锁节点ID列表，如 ["cheetah_A", "cheetah_B2"]
}
```

### 技能树UI交互流程

```
1. 玩家点击已放置的动物塔
2. 弹出技能树面板（居中或塔旁弹出）
3. 显示3个节点：已解锁=高亮，可解锁=正常+费用，不可解锁=灰色
4. 玩家点击可解锁节点：
   a. 检查 player.gold >= node.cost
   b. 检查前置节点已解锁
   c. 检查互斥节点未解锁
   d. 扣除金币，标记节点为已解锁
   e. 调用 apply_skill_effect(tower, node)
5. 面板关闭（点击外部或按ESC）
```

### 技能效果应用

```
apply_skill_effect(tower, node):
  根据 node.effect_type 执行：
    STAT_MODIFIER：直接修改塔的运行时属性（atk_speed, base_atk等）
    ADD_ON_HIT：注册命中回调（如减速、破甲）
    SUMMON：修改狼群子单位数量
    AOE_CHANGE：修改攻击范围参数
    SPECIAL：注册特殊逻辑（群殴判定、嘲讽光环）
```

## Formulas（公式）

**猎豹B1暴击伤害**
```
暴击触发概率：20%
暴击伤害 = base_dmg × 2.0（无视护甲计算后再翻倍）
期望DPS提升 = 1 + 0.2 × (2.0 - 1) = 1.2 → DPS提升20%
```

**鬣狗B2破甲叠加**
```
破甲：target.armor × 0.5，持续5秒
若目标已有破甲：刷新计时，不叠加（armor不会低于armor×0.5）
破甲期间受到伤害计算使用减半后的armor值
```

**狼群B2群殴触发**
```
条件：同帧内 ≥3只狼命中同一目标
群殴伤害 = 每只狼 base_dmg × 1.5（替换原伤害，非追加）
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 金币不足时点击节点 | 扣费检查失败，节点不解锁，提示「金币不足」 |
| 同一局内同一动物放置两只，各自有独立技能树 | 每个 tower_id 对应独立的 TowerSkillState，互不影响 |
| 已选B1后点击B2 | 检查到B2与B1互斥，拒绝解锁，提示「已选择另一路线」 |
| 动物塔被三角龙推走后技能树状态 | 技能绑定在 tower_id 上，推走不影响已解锁的效果 |
| 波次进行中能否升级技能树 | 允许，玩家可随时升级（用战斗中积累的金币） |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| 节点A费用 | 30~60g | 20~80g | 越贵玩家更晚能升级，前期压力更大 |
| 节点B费用 | 50~80g | 40~100g | 越贵Build完成度越低，策略选择更谨慎 |
| 猎豹暴击率 | 20% | 10~35% | 越高伤害越不稳定但峰值越高 |
| 大象踩踏范围A升级 | +1格 | +1~+2格 | 影响AOE覆盖面积和对群体敌人的清场效率 |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 点击已放置的动物塔弹出技能树面板 | 放置猎豹后点击，确认弹出显示3个节点的面板 |
| AC-2 | 节点A可直接购买，B1/B2需先购买A | 未购买A时点击B1，确认被拒绝；购买A后点击B1，确认成功 |
| AC-3 | B1和B2互斥，选一后另一变灰 | 购买B1后，确认B2节点灰色不可点击 |
| AC-4 | 购买节点立即扣除金币 | 购买猎豹A（40g），确认金币减少40 |
| AC-5 | 猎豹解锁B1后攻击有20%概率触发暴击 | 记录猎豹100次攻击，确认暴击次数约20次，暴击伤害为普通×2.0 |
| AC-6 | 两只猎豹各自有独立技能树 | 放置两只猎豹，对一只升级A，确认另一只未升级 |
| AC-7 | 鬣狗解锁B2后攻击降低目标armor50% | 腕龙armor=40%，被鬣狗命中后确认armor变为20%，5秒后恢复 |
