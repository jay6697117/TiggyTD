---

## Overview（概述）

敌人AI系统驱动所有史前生物单位的行为：沿路径移动、攻击动物塔、触发特殊能力。系统读取敌人数据库的ability标签，在运行时执行对应行为逻辑。普通敌人遵循「移动→攻击→特殊能力」的固定循环；Boss拥有多阶段状态机，HP降至阈值时切换阶段并解锁新行为。

---

## Player Fantasy（玩家幻想）

每种史前生物都应该让玩家感受到独特的威胁感。迅猛龙的突然加速让玩家心跳加速；三角龙冲撞推开塔的瞬间让玩家措手不及；菊石群五路分散让玩家AOE不够用。玩家看到新敌人时会下意识判断威胁类型并调整策略——这种「读懂敌人→针对应对」的循环是策略深度的核心。Boss战是章节高潮：阶段切换的视觉冲击和行为变化让玩家感受到史诗级对决。

## Detailed Design（详细设计）

### 普通敌人行为循环

```
每帧更新：
1. 移动：沿路径点序列前进（见寻路系统）
2. 攻击检测：若路径两侧range=1格内有动物塔 → 攻击
3. 特殊能力：计时器触发 → 执行ability逻辑
4. 到达基地 → 对基地造成伤害，自身消除
```

### 攻击行为

```
攻击目标：路径两侧1格内的动物塔
攻击间隔：1.0秒/次（固定，不受atk_speed影响——敌人无atk_speed字段）
攻击伤害：EnemyDB.get(id).atk（输入动物塔的HP计算）
攻击时移动：攻击时停止移动，攻击动画结束后恢复移动
```

### 特殊能力定义

| 能力ID | 所属敌人 | 触发条件 | 效果 |
|--------|---------|---------|------|
| dash | 迅猛龙 | 每5秒 | speed×2，持续3秒，结束后恢复原速 |
| charge | 三角龙 | 检测到前方1格有塔 | 直线冲刺，推开塔1格（通知动物塔放置系统） |
| fly | 翼龙 | 持续被动 | 无视地面陷阱；只被range≥3的单位攻击 |
| trample | 腕龙 | 持续被动 | 每帧对路径两侧1格内的塔造成atk×0.1伤害（溅射） |
| bleed | 剑齿虎 | 每次攻击时 | 附加流血：3秒内每秒造成atk×0.3额外伤害 |
| tusk_strike | 猛犸象 | 每8秒 | 对正前方第一个单位造成atk×2.0伤害 |
| ice_armor | 冰河巨熊 | 被击中时 | 20%概率该次伤害×0.5 |
| steel_bite | 邓氏鱼 | 每次攻击时 | 攻击无视目标20%护甲（伤害计算时armor_rate-0.2，最低0） |

## Formulas（公式）

**流血伤害（bleed）**
```
每秒流血伤害 = atk × 0.3
持续时间 = 3秒
总流血伤害 = atk × 0.3 × 3 = atk × 0.9
示例：剑齿虎atk=30 → 总流血伤害=27（相当于额外0.9次普通攻击）
```

**tusk_strike伤害**
```
技能伤害 = atk × 2.0
示例：猛犸象atk=35 → tusk_strike伤害=70
CD=8秒，普通攻击间隔1秒 → 8秒内造成8次普通+1次技能=8×35+70=350伤害
```

**溅射伤害（trample）**
```
每帧溅射 = atk × 0.1
60fps下每秒溅射 = atk × 0.1 × 60 = atk × 6
注：溅射对象为路径两侧1格内所有塔，每帧各造成一次
```

**dash速度**
```
dash期间speed = base_speed × 2
迅猛龙示例：base_speed=4.0 → dash speed=8.0格/秒
dash持续3秒，额外移动距离 = (8.0-4.0)×3 = 12格
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 翼龙被range=2的塔攻击 | 判定为无效攻击，伤害不生效（fly被动） |
| charge推开目标格非法 | 塔被摧毁（见动物塔放置系统） |
| swarm中一只死亡 | 其余4只继续独立行动，不影响其他个体 |
| Boss无敌护盾期间受伤 | 所有伤害归零（含DOT流血），「纪元审判」除外 |
| 沧龙霸主传送到已有其他敌人的水域格 | 随机重选目标水域，最多重试3次；无可用格则取消传送 |
| Boss阶段切换时正在执行技能 | 当前技能动画播放完毕后再切换阶段状态 |
| ice_armor触发概率与随机性系统 | 使用rng_misc.uniform(0,1) < 0.2判定 |
| dash期间被减速 | 实际speed = base_speed × 2 × (1-减速率)，两者叠加 |
| 敌人到达基地时正在攻击塔 | 中断攻击，立即触发基地伤害并消除自身 |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| 普通敌人攻击间隔 | 1.0秒 | 0.5~2.0 | 越快塔HP损耗越快，对塔HP数值要求更高 |
| dash持续时间 | 3秒 | 2~5秒 | 越长迅猛龙穿越防线的概率越高 |
| dash冷却 | 5秒 | 3~8秒 | 越短迅猛龙几乎全程加速，越长只是偶尔冲刺 |
| tusk_strike冷却 | 8秒 | 5~12秒 | 越短猛犸象爆发越强，越长技能存在感越低 |
| ice_armor触发概率 | 20% | 10~35% | 越高巨熊越肉；超过40%玩家会感到伤害随机性过强 |
| Boss阶段2召唤援军数量 | 2只迅猛龙 | 1~4只 | 越多波次压力越大，超过3只可能导致玩家无法应对 |
| 沧龙霸主传送CD | 15秒 | 10~25秒 | 越短Boss越难追击，越长水域传送战术价值降低 |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 所有敌人沿路径点移动，到达基地后消除并造成伤害 | 放空防线，确认敌人到达基地后消失并基地HP减少 |
| AC-2 | 迅猛龙每5秒触发dash，速度翻倍持续3秒 | 计时迅猛龙移动速度，确认每5秒出现3秒的速度×2 |
| AC-3 | 三角龙charge推开前方塔 | 在三角龙路径前放塔，确认冲撞后塔被推开1格 |
| AC-4 | 翼龙无视地面陷阱 | 在路径上放陷阱，确认翼龙经过时陷阱不触发 |
| AC-5 | 翼龙只被range≥3的塔攻击 | range=2的塔不攻击翼龙，range=3的塔正常攻击翼龙 |
| AC-6 | 剑齿虎攻击附加流血DOT | 剑齿虎攻击塔后，塔HP在3秒内持续减少（atk×0.3/秒） |
| AC-7 | 菊石群5只独立寻路攻击不同目标 | 生成菊石群，确认5只分散攻击而非全部攻击同一塔 |
| AC-8 | 霸王龙王HP≤50%时speed和atk提升 | 将霸王龙王HP打至50%以下，确认移动速度和攻击力增加 |
| AC-9 | 沧龙霸主无敌护盾期间普通伤害归零 | 护盾激活时攻击沧龙霸主，确认HP不减少 |
| AC-10 | 沧龙霸主离开水域时受伤害+100% | 记录沧龙霸主在水域和陆地受到的相同攻击伤害，确认陆地×2 |
