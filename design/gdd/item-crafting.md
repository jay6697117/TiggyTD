# 道具合成系统

> **状态**: 已完成
> **作者**: 开发者 + Claude
> **最后更新**: 2026-03-27

---

## Overview（概述）

道具合成系统允许玩家在商店阶段将两件普通道具合成一件高级道具。合成遵循固定配方（非自由组合），配方由道具数据库定义。合成成功后两件材料道具从背包消失，高级道具进入背包。合成是 Tangy TD 最受玩家好评的机制，核心价值在于让玩家感知到「我在构建一件强力物品」的目标感，以及「凑齐材料」的收集满足感。

---

## Player Fantasy（玩家幻想）

合成系统的高光时刻是「我终于凑齐了」：三波以来一直在等第二件材料，波次奖励终于掉出来，立刻合成出传说级道具然后装备到核心动物上——这种「计划实现」的满足感是长线目标感的来源。合成也创造了独特的决策：现在用这两件普通道具，还是等着凑成更强的合成品？这个权衡让每局的节奏感不同。

## Detailed Design（详细设计）

### 合成配方（MVP阶段共6个配方）

```
配方1：攻速鼓 + 时间沙漏 → 永恒节拍器（攻速+50%，范围提升）
配方2：荆棘甲 + 钢铁护盾 → 镜像盔甲（反弹20%伤害+护甲+30%）
配方3：毒牙 + 破甲矛 → 腐蚀巨刃（中毒+破甲双效，各强化50%）
配方4：生命水晶 + 再生之泉 → 永生图腾（HP上限+50，每秒回复3HP）
配方5：攻速鼓 + 破甲矛 → 战场霸主（攻速+20%+攻击附带破甲）
配方6：荆棘甲 + 毒牙 → 毒刺盾（受击反弹中毒效果给攻击者）
```

### 合成UI流程

```
合成入口：商店界面右侧「合成台」区域

流程：
1. 玩家打开商店（build阶段或波次间）
2. 点击「合成台」标签
3. 显示所有配方列表（可合成的高亮，缺材料的灰色显示缺少哪件）
4. 玩家点击高亮配方 → 弹出确认对话框：
   「消耗 [材料A] + [材料B]，合成 [结果道具]？」
5. 确认 → 材料从背包移除，结果道具加入背包
6. 取消 → 无变化
```

### 合成验证逻辑

```
can_craft(recipe):
  return inventory.contains(recipe.material_A)
    AND inventory.contains(recipe.material_B)
    AND inventory.free_slots >= 1  // 合成后净减1格（2个材料→1个结果）

execute_craft(recipe):
  inventory.remove(recipe.material_A)
  inventory.remove(recipe.material_B)
  result = ItemDatabase.create(recipe.result_id)
  inventory.add(result)
```

### 材料匹配规则

```
材料匹配基于 item_id（精确匹配），不是道具类型
同一道具在背包中有多件时，使用第一个匹配的实例
道具必须在背包中（已装备的道具不参与合成）
```

## Formulas（公式）

**合成收益评估**
```
合成品价值 ≈ 材料A价值 + 材料B价值 + 合成溢价
合成溢价目标：+30~50%（即合成品效果强于两件普通道具之和30~50%）
示例：攻速鼓(+20%) + 时间沙漏(+15%) = 普通叠加+35%
      永恒节拍器 = +50%攻速（溢价约43%）
```

**背包空间计算**
```
合成前：背包占用 N 格（含两件材料）
合成后：背包占用 N-1 格（两件变一件）
特殊情况：若背包已满（6格），执行合成仍可进行（净减1格，不会溢出）
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 材料已装备到单位上，不在背包中 | 合成检查只查背包，装备中的道具不可作为材料 |
| 背包满（6格）时尝试合成 | 合成后净减1格（变5格），允许合成，无需额外空间 |
| 同一道具是多个配方的材料 | 玩家选择哪个配方就消耗对应材料，不自动决定 |
| 两件完全相同的道具能否合成 | MVP阶段没有「同种×2」配方，无此情况 |
| 合成后背包中恰好有两件结果道具（已有一件同类） | 允许，背包中可有多件同名道具（道具实例独立） |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| MVP合成配方数量 | 6个 | 4~10个 | 越多组合可能性越丰富，但配方记忆负担越重 |
| 合成溢价幅度 | +30~50% | +20~70% | 越高合成品越值得追求，玩家愿意等待凑材料 |
| 材料道具掉落率 | （由商店系统控制） | — | 影响凑齐材料的难度和节奏 |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 背包中有两件材料时配方高亮可点击 | 背包放入攻速鼓+时间沙漏，确认配方1高亮 |
| AC-2 | 缺少材料时配方灰色并提示缺少哪件 | 只有攻速鼓，确认配方1灰色并显示「缺少：时间沙漏」 |
| AC-3 | 合成后两件材料从背包消失，结果道具出现 | 合成永恒节拍器，确认攻速鼓和时间沙漏消失，背包出现永恒节拍器 |
| AC-4 | 已装备的道具不参与合成材料检查 | 攻速鼓已装备到猎豹，合成台确认配方1灰色（材料不足） |
| AC-5 | 合成后背包格子净减1 | 6格背包装2件材料（占2格），合成后占1格（净减1） |
| AC-6 | 确认对话框正确显示材料和结果名称 | 点击配方1确认，弹窗显示「消耗 攻速鼓 + 时间沙漏，合成 永恒节拍器？」 |
