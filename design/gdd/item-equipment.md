# 道具装备系统

> **状态**: 已完成
> **作者**: 开发者 + Claude
> **最后更新**: 2026-03-27

---

## Overview（概述）

道具装备系统允许玩家将从商店购买或战场拾取的道具装备到英雄或动物塔上，提供被动属性加成或特殊效果。英雄最多装备3件道具，每只动物塔最多装备1件道具。道具装备后在当前局内永久生效，不可转移给其他单位。道具数据库定义所有道具属性，本系统管理装备槽位、效果应用和UI展示。

---

## Player Fantasy（玩家幻想）

道具装备是「搭配出神奇组合」的乐趣来源。把「攻速鼓」装到猎豹塔上、再叠上技能树的暴击节点，看着猎豹进入狂暴输出模式——这种「1+1>2」的组合感是 Build 深度的直觉体现。英雄身上的3件道具代表这一局的战略方向，是玩家向朋友炫耀「我这局怎么赢的」的故事核心。

## Detailed Design（详细设计）

### 装备槽位

```
英雄：3个装备槽
动物塔：1个装备槽（每只塔独立）
装备后不可移除（局内永久）
装备上限满时无法继续装备新道具
```

### 道具装备流程

```
玩家获得道具（来源：商店购买 / 波次奖励）：
  道具进入「背包」（最多6格，溢出报警）

装备操作：
  1. 玩家点击背包中的道具
  2. 进入「装备目标选择」模式（鼠标变为装备图标）
  3. 玩家点击目标单位（英雄或动物塔）
  4. 检查目标装备槽是否已满
  5. 装备成功：道具从背包移入目标装备槽，效果立即生效
  6. 玩家按ESC取消装备模式（道具留在背包）
```

### 效果应用

```
apply_item_effect(unit, item):
  根据 item.effect_type：
    STAT_MODIFIER：直接修改单位运行时属性
    ON_HIT：注册命中回调
    ON_KILL：注册击杀回调
    AURA：注册持续光环（装备方不同于英雄光环，单独层级）
    ACTIVE：（MVP不实现，Vertical Slice后考虑）
```

### 背包管理

```
背包容量：6格
超出时：新道具无法进入背包，提示「背包已满，请先装备或丢弃道具」
丢弃：玩家可将背包中道具拖出丢弃（不可回收，不退金币）
```

## Formulas（公式）

**道具属性叠加**
```
同类STAT_MODIFIER叠加方式：加法叠加
示例：攻速鼓(+20% atk_speed) + 时间沙漏(+15% atk_speed) = +35% atk_speed
最终值 = base × (1 + Σ所有加成百分比)
```

**道具与技能树叠加**
```
道具加成与技能树加成独立加法叠加后，再与光环乘法叠加
示例：猎豹 base_atk_speed=2.5
  技能树A节点 +15% → ×1.15
  装备攻速鼓 +20% → 加法叠加后 ×1.35
  英雄光环 ×1.2（乘法）
  最终：2.5 × 1.35 × 1.2 = 4.05
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 背包满时获得新道具 | 新道具不进入背包，提示「背包已满」；道具丢失（不进入游戏世界） |
| 动物塔已有装备时尝试再装一件 | 检查失败，提示「该单位装备槽已满」 |
| 英雄3槽都满时尝试装备 | 检查失败，提示「英雄装备已满」 |
| 道具装备到被三角龙推走的塔上 | 道具效果随塔移动，塔不死亡则效果持续 |
| 同一道具能否装备到多只动物塔 | 可以（不同道具实例），背包中有几件就能装几只塔 |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| 英雄装备槽数量 | 3 | 2~4 | 越多英雄Build深度越高，道具价值越大 |
| 动物塔装备槽数量 | 1 | 1~2 | 若为2则塔的Build深度大幅提升，复杂度也增加 |
| 背包容量 | 6格 | 4~8格 | 越小玩家决策压力越大（必须快速装备或丢弃） |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 从背包装备道具到动物塔，效果立即生效 | 将攻速鼓装备到猎豹，确认猎豹攻速立即提升20% |
| AC-2 | 动物塔装备槽满时拒绝第二件道具 | 猎豹已装备一件，再次尝试装备，确认提示「装备槽已满」 |
| AC-3 | 英雄3件装备槽全满时拒绝第四件 | 英雄装满3件后尝试装第四件，确认被拒绝 |
| AC-4 | 背包超过6格时新道具无法进入 | 背包已有6件道具，购买第7件，确认提示「背包已满」 |
| AC-5 | 道具tooltip正确显示效果描述 | 鼠标悬停道具图标，确认弹出tooltip含名称、稀有度、效果 |
| AC-6 | 道具与技能树加成正确叠加 | 猎豹技能树A(+15%)+攻速鼓(+20%)，确认总攻速=base×1.35 |
