# 英雄光环系统

> **状态**: 已完成

> **最后更新**: 2026-03-27
> **实现支柱**: 支柱一（图腾指挥感——英雄存在本身就是战场增益）

---

## Overview（概述）

英雄光环系统提供老虎英雄的被动持续光环效果：只要英雄存活且在场，光环就持续生效于其周围一定范围内的动物塔。光环不需要主动触发，每帧检测英雄位置并更新受益塔列表。当英雄移动时，光环跟随移动，离开范围的塔失去加成，进入范围的塔获得加成。光环是英雄走位策略的核心驱动力——去哪里比放什么塔同样重要。

---

## Player Fantasy（玩家幻想）

光环让英雄从「可操控的单位」变成「移动的圣地」。玩家操控老虎在战场上奔跑，看到经过的动物塔突然发光、输出暴涨——这种「英雄就是胜负手」的感觉是图腾指挥感的精髓。战况危急时冲向压力最大的防线、用光环顶住关键波次的冲击，是最让人上头的操作时刻。光环的存在让「英雄站在哪里」成为有意义的战术决策。

## Detailed Design（详细设计）

### 光环属性

| 属性 | 值 |
|------|----|
| 光环类型 | 被动持续（无需激活） |
| 光环半径 | 3格（圆形，以英雄中心） |
| 效果 | 范围内动物塔 atk_speed +20% |
| 更新频率 | 每0.1秒重新计算受益塔列表（非每帧，节省性能） |
| 视觉 | 英雄周围持续金色光圈；受益塔有轻微金色光晕 |

### 光环更新逻辑

```
每0.1秒：
1. 以英雄当前位置为中心，查询半径3格内所有 TOWER 层单位
2. 新受益列表 = 查询结果
3. 对比上次受益列表：
   - 新进入的塔：apply_aura_buff(tower, +20% atk_speed)
   - 离开范围的塔：remove_aura_buff(tower)
   - 持续在范围内的塔：维持buff，不重复施加
4. 更新 last_benefited_towers = 新受益列表
```

### 光环叠加规则

```
光环与英雄技能（百兽怒吼/自然之召）独立叠加：
  - 光环：atk_speed × 1.2（被动）
  - 百兽怒吼激活时：atk_speed × 1.2 × 1.5 = atk_speed × 1.8
  - 两者共同作用时使用乘法叠加
光环标签："hero_aura"，每个塔只能有一个hero_aura层
（英雄只有一个，不存在多重光环问题）
```

### 英雄死亡处理

```
MVP阶段：英雄不会死亡（无死亡机制）
光环始终有效
（Alpha阶段如引入英雄HP：死亡时清除所有光环buff）
```

## Formulas（公式）

**光环攻速加成**
```
加成后攻速 = base_atk_speed × 1.2
示例：猎豹 atk_speed=2.5 → 光环加成后=3.0（每0.333秒攻击一次）
示例：大象 atk_speed=0.5 → 光环加成后=0.6
```

**光环+百兽怒吼叠加**
```
同时生效时：atk_speed × 1.2 × 1.5 = atk_speed × 1.8
示例：猎豹 → 2.5 × 1.8 = 4.5（每0.222秒攻击一次）
```

**光环覆盖范围**
```
光环半径 = 3格
覆盖面积 ≈ π × 3² ≈ 28格
（典型地图20×15=300格，光环覆盖约9%地图面积）
```

**更新频率性能估算**
```
每0.1秒更新一次（10次/秒）
每次圆形查询复杂度 = O(塔数量) ≈ O(20)（最多20只塔）
性能影响：可忽略
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 英雄在光环范围边界快速来回移动 | 0.1秒更新间隔天然防抖，不会每帧切换buff状态 |
| 光环范围内塔被三角龙推出范围 | 下次0.1秒更新时检测到塔不在范围内，自动移除buff |
| 新放置的塔恰好在光环范围内 | 下次0.1秒更新时检测到新塔，自动施加buff |
| 光环buff与状态效果系统的减速叠加 | 独立计算：atk_speed先乘光环加成，再受减速影响（减速作用于最终值） |
| 英雄施放自然之召时光环同时生效 | 两者独立：光环buff=atk_speed×1.2，自然之召buff=base_atk×1.3，不冲突 |
| 狼群3只子单位是否各自受光环影响 | 是，狼群每只子单位作为独立塔单位，各自接受光环buff |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| 光环半径 | 3格 | 2~5格 | 越大英雄走位压力越小，越小要求精确站位 |
| 光环攻速加成 | +20%（×1.2） | ×1.1~×1.4 | 越高英雄存在感越强，超过×1.5可能让固定塔失去意义 |
| 光环更新频率 | 0.1秒 | 0.05~0.2秒 | 越小响应越灵敏但性能开销增加；越大有明显「迟滞感」 |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 光环半径3格内的塔攻速提升20% | 将英雄站在猎豹旁，确认猎豹攻击间隔从0.4秒变为约0.333秒 |
| AC-2 | 光环半径外的塔不受影响 | 英雄站在地图一端，确认另一端的塔攻速不变 |
| AC-3 | 英雄移动时光环跟随，离开范围的塔失去buff | 英雄从塔旁走开超过3格，确认该塔攻速恢复正常 |
| AC-4 | 新进入范围的塔自动获得buff | 英雄走近一个原本在范围外的塔，确认该塔攻速提升 |
| AC-5 | 光环与百兽怒吼叠加为×1.8 | 光环生效时触发百兽怒吼，确认猎豹攻速=2.5×1.8=4.5 |
| AC-6 | 狼群3只子单位各自受光环影响 | 英雄站在狼群旁，确认3只子单位均攻速提升 |
| AC-7 | 受光环影响的塔有金色光晕视觉反馈 | 确认范围内塔有光晕，范围外塔无光晕 |
| AC-8 | 三角龙将塔推出范围后塔失去buff | 三角龙将光环范围内的塔推出3格外，确认0.1秒内塔攻速恢复正常 |
