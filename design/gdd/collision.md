# 碰撞检测系统

> **状态**: 设计中
> **作者**: 开发者 + Claude
> **最后更新**: 2026-03-27
> **实现支柱**: 技术基础（支撑所有战斗系统）

---

## Overview（概述）

碰撞检测系统是 TiggyTD 所有物理交互的技术基础。它负责三类检测：①单位边界碰撞（英雄移动、塔放置合法性）、②攻击命中判定（近战范围检测、远程投射物飞行）、③技能效果范围检测（AOE、光环）。系统输出标准化的碰撞事件，供伤害计算、状态效果等上层系统消费，本系统不包含任何游戏逻辑。

---

## Player Fantasy（玩家幻想）

碰撞系统是「感觉对了」的基础设施。玩家不会直接感知到它，但会通过以下方式感受到它的质量：鹰射出的羽箭准确命中迅猛龙头部、犀牛的反击即时生效、老虎在地图上流畅移动不卡墙角。碰撞精度直接影响游戏的「手感」——不准确的碰撞会让玩家感到「明明打中了却没伤害」。

## Detailed Design（详细设计）

### Core Rules（核心规则）

#### 碰撞体类型

| 对象 | 碰撞体类型 | 说明 |
|------|-----------|------|
| 英雄（老虎） | 圆形 | 半径0.4格，用于移动边界和技能范围 |
| 动物塔 | 圆形 | 半径0.45格，用于放置合法性和被攻击判定 |
| 敌人单位 | 圆形 | 半径按单位大小：小型0.3格，中型0.45格，大型0.6格 |
| 地形边界 | 矩形 | 格子地图边界，英雄和单位不可穿越 |
| 建造区域 | 矩形 | 可放置塔的格子范围 |
| 攻击射程 | 圆形 | 以塔为圆心，`range`值为半径 |
| 投射物 | 圆形 | 半径0.1格，飞行中检测命中 |
| AOE效果 | 圆形 | 以技能释放点为圆心，效果半径由技能定义 |

#### 命中判定规则

**近战塔（range ≤ 2.0格）：范围检测**
```
每次攻击间隔到达时：
1. 以塔位置为圆心，range为半径，检测范围内所有敌人
2. 按目标选择策略选定目标（默认：最近的敌人）
3. 立即触发命中事件（无飞行时间）
```

**远程塔（range > 2.0格）：投射物**
```
每次攻击间隔到达时：
1. 以塔位置为圆心，range为半径，检测范围内所有敌人
2. 选定目标后，生成投射物朝目标飞行
3. 投射物speed = 10格/秒（固定）
4. 投射物追踪目标（归巢式），目标死亡时投射物消失
5. 投射物碰撞体与敌人碰撞体重叠时触发命中事件
```

**英雄技能范围检测**
```
技能释放时：
1. 以释放点为圆心，技能半径为范围，瞬时检测所有敌人
2. 所有在范围内的敌人同时触发命中事件
3. AOE技能不使用投射物
```

**翼龙特殊规则**
```
翼龙的碰撞层级 = AIR_LAYER
地面陷阱碰撞层级 = GROUND_LAYER
AIR_LAYER 与 GROUND_LAYER 不发生碰撞
只有 range ≥ 3.0 的塔才能攻击 AIR_LAYER 单位
```

#### 碰撞层级定义

| 层级 | 说明 | 与谁碰撞 |
|------|------|----------|
| HERO | 英雄老虎 | TERRAIN, ENEMY |
| TOWER | 动物塔 | TERRAIN, ENEMY（被攻击时） |
| ENEMY_GROUND | 地面敌人 | TERRAIN, TOWER_ATTACK, TRAP |
| ENEMY_AIR | 飞行敌人（翼龙） | TOWER_ATTACK_RANGE（range≥3） |
| PROJECTILE | 投射物 | ENEMY_GROUND, ENEMY_AIR |
| TRAP | 陷阱 | ENEMY_GROUND（不含AIR） |
| TERRAIN | 地形边界 | HERO, ENEMY_GROUND |

### States and Transitions（状态与转换）

| 状态 | 说明 |
|------|------|
| IDLE | 无碰撞检测请求 |
| DETECTING | 执行范围检测或投射物飞行检测 |
| HIT_RESOLVED | 命中事件已生成，等待上层系统消费 |

碰撞检测每帧运行，物理引擎（Godot Physics）自动处理；投射物飞行在每帧 `_physics_process` 中更新位置。

### Interactions with Other Systems（与其他系统的交互）

| 系统 | 交互方式 |
|------|----------|
| 伤害计算系统 | 输出命中事件（攻击方ID、目标ID、攻击类型标签） |
| 英雄移动系统 | 提供TERRAIN和TOWER碰撞体，阻止英雄穿越 |
| 动物塔放置系统 | 提供建造区域碰撞体，验证放置合法性 |
| 敌人AI系统 | 提供TERRAIN碰撞体用于寻路边界 |
| 动物塔攻击系统 | 接收范围检测结果，决定攻击目标 |
| 英雄技能系统 | 接收AOE范围检测结果 |
| 英雄光环系统 | 持续检测光环范围内的友军单位 |
| 状态效果系统 | 减速效果影响投射物追踪速度（敌人减速不影响投射物，只影响敌人位移） |

## Formulas（公式）

**圆形碰撞重叠检测**
```
距离 = sqrt((x2-x1)² + (y2-y1)²)
碰撞 = 距离 < (半径1 + 半径2)
```

**投射物飞行时间**
```
飞行时间（秒）= 当前距离 / 投射物speed(10格/秒)
// 例：鹰攻击7格外目标，飞行时间 = 7/10 = 0.7秒
```

**AOE命中范围**
```
命中条件：目标圆心到AOE圆心的距离 < AOE半径 + 目标碰撞半径
```

**光环持续检测（每帧）**
```
友军在光环内 = 友军圆心到英雄圆心的距离 < 光环半径
// 光环半径由英雄光环系统定义
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 投射物目标在飞行途中死亡 | 投射物立即消失，不命中其他敌人 |
| 多个投射物同帧命中同一目标 | 每个投射物各自触发独立命中事件，均正常结算 |
| 翼龙与地面塔攻击范围重叠 | range<3的塔不触发命中事件（层级不匹配） |
| 英雄移动至塔占据的格子 | 英雄碰撞体与塔碰撞体不重叠（不同层级），英雄可在塔旁边移动 |
| 大型敌人（腕龙）碰撞体超出路径 | 腕龙碰撞半径0.6格，路径宽度至少1.5格，不会卡路径 |
| 咆哮石AOE与光环同帧检测 | 各自独立检测，互不干扰 |
| 三角龙冲撞推开塔时碰撞体重叠 | 推开动画期间临时禁用塔碰撞体，动画结束后重新启用 |
| 灵魂链接两塔距离判定 | 使用碰撞检测：两塔碰撞体扩展至1.5格半径，重叠即为「相邻」 |

### 命中事件数据结构

```
HitEvent {
  attacker_id: String    // 攻击方单位ID
  target_id: String      // 目标单位ID

}
```

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| 投射物飞行速度 | 10格/秒 | 7~15 | 低于7秒远程攻击延迟感明显；高于15视觉来不及看清 |
| 近远程塔分界range | 2.0格 | 1.5~2.5 | 影响哪些塔使用投射物视觉效果 |
| 英雄碰撞半径 | 0.4格 | 0.3~0.5 | 太大英雄容易卡地形；太小穿越感不真实 |
| 大型敌人碰撞半径 | 0.6格 | 0.5~0.8 | 影响腕龙/Boss的命中判定宽容度 |
| 灵魂链接相邻判定半径 | 1.5格 | 1.0~2.0 | 影响灵魂链接的激活距离 |
| 光环检测频率 | 每帧 | 每帧/每5帧 | 每5帧检测可节省性能但光环边界不精确 |

## Acceptance Criteria（验收标准）

| # | 标准 | 验证方法 |
|---|------|----------|
| AC-1 | 近战塔（range≤2）攻击即时命中，无飞行延迟 | 犀牛攻击相邻敌人，伤害数字与攻击动画同帧出现 |
| AC-2 | 远程塔（range>2）发射投射物飞向目标 | 鹰攻击7格外目标，可见投射物飞行约0.7秒后命中 |
| AC-3 | 投射物目标死亡时投射物消失 | 目标被其他塔击杀，飞行中的投射物立即消失 |
| AC-4 | 翼龙不被range<3的塔攻击 | 翼龙经过犀牛（range=1.5）攻击范围，犀牛不触发攻击 |
| AC-5 | 翼龙被range≥3的鹰正常攻击 | 翼龙进入鹰（range=7）攻击范围，鹰正常发射投射物 |
| AC-6 | 英雄无法穿越地形边界 | 操控老虎向地图边界移动，确认被阻挡 |
| AC-7 | AOE技能同时命中范围内所有敌人 | 释放AOE技能，范围内3只敌人同帧各自触发命中事件 |
| AC-8 | 灵魂链接在两塔≤1.5格时激活 | 将两只同类动物塔放置于相邻格，确认灵魂链接激活 |
| AC-9 | 命中事件包含正确的攻击类型标签 | 近战攻击=melee，投射物=projectile，AOE=aoe |
| AC-10 | 同帧多个投射物命中同一目标各自独立结算 | 3个投射物同帧命中同一敌人，触发3次独立伤害计算 |
