## Overview（概述）

英雄技能系统管理老虎英雄的三个主动技能：百兽怒吼（Apex Roar）、自然之召（Nature's Call）、纪元审判（Era Judgement）。技能通过独立冷却计时器控制使用频率，玩家点击技能按钮或快捷键触发。技能效果覆盖范围检测依赖碰撞检测系统，伤害输出调用伤害计算系统，状态附加调用状态效果系统。英雄处于CASTING状态时无法移动。

---

## Player Fantasy（玩家幻想）

英雄技能是「我是指挥官」的最直接体验。百兽怒吼在关键时刻让全场动物爆发，看着伤害数字暴涨的瞬间是「我的Build奏效了」的巅峰感。自然之召让英雄变成战场纽带，追着危机点跑去放光环的操作感是塔防游戏里罕见的主动参与感。纪元审判对Boss的致命一击是章节高潮——护盾碎裂、Boss倒下，那一刻是整局的情绪顶点。

## Detailed Design（详细设计）

### 技能概览

| 技能 | 快捷键 | 冷却 | 施法时间 | 范围 | 核心效果 |
|------|--------|------|---------|------|----------|
| 百兽怒吼（Apex Roar） | Q | 30秒 | 0.5秒 | 全场 | 所有动物塔攻速+50%，持续8秒 |
| 自然之召（Nature's Call） | W | 45秒 | 0.8秒 | 以英雄为中心5格圆形 | 范围内动物塔获得「激励」：atk+30%，持续10秒 |
| 纪元审判（Era Judgement） | E | 120秒 | 1.5秒 | 目标单体（点击选择） | 对目标造成300固定伤害，无视护甲；打破沧龙霸主无敌护盾 |

### 技能触发流程

```
玩家点击技能按钮 / 按快捷键：
1. 检查冷却是否就绪（cooldown_timer ≤ 0）
2. 技能需要选择目标（纪元审判）→ 进入TARGET_SELECT模式，等待玩家点击目标
3. 英雄进入CASTING状态（停止移动）
4. 播放施法动画（casting_time秒）
5. 施法完成 → 执行技能效果
6. 重置冷却计时器（cooldown_timer = max_cooldown）
7. 英雄返回IDLE/MOVING状态
```

### 百兽怒吼（Apex Roar）

```
效果：全场所有已放置动物塔的 atk_speed × 1.5
持续时间：8秒
实现：向所有塔单位广播 apply_buff("apex_roar", duration=8)
结束时：atk_speed 恢复原值（不叠加多次广播）
视觉：英雄播放怒吼动画，全场塔出现金色光效
```

### 自然之召（Nature's Call）

```
效果：以英雄当前位置为中心，5格圆形范围内所有动物塔 base_atk × 1.3
持续时间：10秒
实现：碰撞检测系统查询 hero_position 半径5格内的 TOWER 层单位
       对每个命中塔调用 apply_buff("natures_call", duration=10)
结束时：base_atk 恢复原值
视觉：英雄位置绿色光环扩散，命中塔出现树叶粒子效果
英雄移动不影响已施加的buff（施法时锁定范围）
```

### 纪元审判（Era Judgement）

```
效果：
  1. 对选定目标造成300点固定伤害（不经过伤害计算系统，直接扣HP）
  2. 若目标为沧龙霸主且处于无敌护盾状态 → 强制打破护盾
施法流程：
  1. 进入TARGET_SELECT模式（光标变为准星图标）
  2. 玩家点击敌人目标 → 确认目标
  3. 若点击非敌人区域 → 取消施法，技能不进入冷却
  4. 播放施法动画1.5秒
  5. 发射光柱打击目标
  6. 进入冷却
视觉：金色光柱从英雄打向目标，巨大伤害数字（金色）
```

### 冷却管理

```
每帧：cooldown_Q -= delta, cooldown_W -= delta, cooldown_E -= delta
冷却中：技能按钮变灰，显示剩余冷却秒数
就绪：技能按钮高亮
施法中（CASTING）：所有技能按钮锁定，无法触发新技能
```

### 状态机

| 状态 | 说明 |
|------|------|
| IDLE | 无施法操作 |
| TARGET_SELECT | 等待玩家选择目标（纪元审判专用） |
| CASTING | 施法动画播放中，英雄停止移动 |

## Formulas（公式）

**百兽怒吼攻速加成**
```
加成后攻速 = base_atk_speed × 1.5
示例：猎豹 atk_speed=2.5 → 加成后=3.75（每0.267秒攻击一次）
持续时间：8秒
```

**自然之召攻击加成**
```
加成后攻击 = base_atk × 1.3
示例：老虎 base_atk=60 → 加成后=78
范围：hero_position 半径5格圆形
持续时间：10秒
```

**纪元审判伤害**
```
伤害 = 300（固定，无视护甲，不经过 DamageSystem.calculate）
额外效果：打破沧龙霸主无敌护盾
```

**冷却时间**
```
Apex Roar CD = 30秒
Nature's Call CD = 45秒
Era Judgement CD = 120秒
```

## Edge Cases（边界情况）

| 情况 | 处理方式 |
|------|----------|
| 百兽怒吼期间再次施放（CD未到） | 按钮锁定，无法触发；buff不重置计时，继续原有持续时间 |
| 自然之召施放时范围内无塔 | 技能正常施放、进入冷却，无buff生效，无提示 |
| 纪元审判目标在施法过程中死亡 | 施法完成后伤害无效（目标不存在）；技能仍进入冷却 |
| 纪元审判点击非敌人区域取消 | 技能不进入冷却，返回就绪状态 |
| CASTING期间被敌人攻击 | 施法不中断（英雄有独立HP），但英雄停止移动 |
| 百兽怒吼与自然之召buff同时生效 | 独立叠加：atk_speed×1.5 且 base_atk×1.3，两者不冲突 |
| 自然之召buff期间塔被摧毁 | buff随塔单位消失，不影响其他塔的buff |
| 英雄在CASTING期间波次结束 | CASTING状态立即中断，技能效果不生效，不进入冷却 |

## Tuning Knobs（调节旋钮）

| 旋钮 | 当前值 | 安全范围 | 影响的游戏体验 |
|------|--------|---------|---------------|
| Apex Roar 攻速加成 | ×1.5 | ×1.2~×2.0 | 越高爆发窗口越强，超过×2.0可能让DPS失衡 |
| Apex Roar 持续时间 | 8秒 | 5~12秒 | 越长覆盖更多波次，越短更像点爆发 |
| Apex Roar CD | 30秒 | 20~45秒 | 越短使用频率越高，影响全局DPS期望值 |
| Nature's Call 范围 | 5格 | 3~7格 | 越大越容易覆盖全场，降低英雄走位的策略性 |
| Nature's Call 攻击加成 | ×1.3 | ×1.1~×1.6 | 越高单次覆盖收益越大，需与CD平衡 |
| Nature's Call CD | 45秒 | 30~60秒 | 越短英雄走位价值越高 |
| Era Judgement 固定伤害 | 300 | 200~500 | 越高对Boss的斩杀线越低，影响Boss战节奏 |
| Era Judgement CD | 120秒 | 90~180秒 | 越短玩家可多次使用，降低Boss战难度 |
